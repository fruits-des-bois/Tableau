# 📊 Projet Streamlit

Une application web développée avec **Python** et **Streamlit** offrant une interface interactive pour visualiser et traiter des données.

## 🚀 Fonctionnalités

- Interface utilisateur intuitive
- Visualisation interactive des données
- Traitement automatisé des informations
- Exécution directement dans le navigateur
- Développée entièrement avec Python et Streamlit

## 📂 Structure du projet

```text
.
├── app_streamlit.py      # Application principale
├── requirements.txt      # Dépendances Python
├── README.md             # Documentation
└── assets/               # Ressources (optionnel)
````

## 🛠️ Prérequis

* Python 3.10 ou supérieur
* pip

## 📥 Installation

Clonez le dépôt :

```bash
git clone https://github.com/votre-utilisateur/votre-projet.git
cd votre-projet
```

Installez les dépendances :

```bash
pip install -r requirements.txt
```

## ▶️ Exécution

Lancez l'application avec :

```bash
streamlit run app_streamlit.py
```

ou

```bash
python -m streamlit run app_streamlit.py
```

Par défaut, l'application sera disponible à l'adresse :

```
http://localhost:8501
```

## 📦 Dépendances principales

* Streamlit
* Pandas
* NumPy
* Matplotlib
* Plotly
